## 3. Сбор запросов по разделам §7 (ru; для uk — зеркалим, конкуренция почти нулевая §3)

Разделение на **транзакционные (T)** и **справочные (S)**. [ДАННЫЕ] = реальные поисковые формулировки/подсказки из ресёрча; частотность НЕ замерялась (нужен GSC/Keyword-tool — KPI §6).

**Достопримечательности (S):** «{место} как добраться», «{место} время работы / цена билета {год}», «что посмотреть в {регионе}», «Вардзия как добраться из Тбилиси», «Уплисцихе цена билета», «каньон Мартвили / Окаце как добраться», «пещера Прометея». [ДАННЫЕ]

**Города (S, частично T):** «{город} что посмотреть за N дней самостоятельно», «{город} как добраться», «{город} где поесть», «{город} где остановиться / жильё» (→T), «Местиа как добраться», «Сигнахи что посмотреть». [ДАННЫЕ]

**Еда (S+директория, T по цене):** «где поесть в {городе}», «лучшие рестораны Тбилиси / Батуми», «что попробовать в Грузии / грузинская кухня», «сколько стоит поесть в Грузии» (→T), «лучшие хинкали/хачапури {город}», «винные дегустации Кахетия». [ДАННЫЕ]

**Маршруты (S, T на концовке):** «Грузия на машине маршрут на неделю / 7 дней», «маршрут Тбилиси Батуми на машине», «Кахетия маршрут на 2 дня», «Местиа Ушгули маршрут / трек», «по Грузии на машине что посмотреть + карта». [ДАННЫЕ]

**Аренда авто (T):** «аренда авто Грузия без депозита / без кредитки», «прокат авто Тбилиси / Батуми / Кутаиси аэропорт цена», «сколько стоит аренда авто в Грузии», «аренда авто Грузия для россиян / украинцев», «Localrent отзывы», «аренда авто депозит развод подвохи». [ДАННЫЕ]

**Страховки (T+S):** «страховка для Грузии обязательная 2026», «нужна ли страховка в Грузию», «SafetyWing отзыв / опыт», «EKTA страховка Грузия» (uk-приоритет), «страховка в Грузию цена / онлайн», «штраф без страховки Грузия». [ДАННЫЕ]

**Новости/сезонное (S):** «когда лучше ехать в Грузию», «Грузия 2026 новые правила въезда», «сезон в Грузии», «Грузия зимой / горнолыжный сезон». [ДАННЫЕ]

**eSIM/связь (T):** «eSIM Грузия купить», «сим-карта Грузия туристу 2026», «какой оператор / интернет в Грузии», «Magti / Airalo Грузия». [ДАННЫЕ]

**uk-зеркало:** те же кластеры по-украински («що подивитися в Грузії», «оренда авто Грузія», «страховка Грузія EKTA»). Ресёрч [ДАННЫЕ]: uk-выдачу держат in-facts.info/uk и tip-to-trip (часто 2021, устаревшее) → **низкая конкуренция по свежему контенту**, приоритет для лёгкого входа.

## 5. Где самый лёгкий вход в топ [ОЦЕНКА]
1. **uk-зеркало любой темы** — конкуренция почти нулевая, многое у конкурентов от 2021 (ekta-dlya-ukraincev, оренда-авто, маршруты uk).
