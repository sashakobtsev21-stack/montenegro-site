- **Интерактивные состояния поиска/фильтров** (пусто/0/загрузка/ошибка/клавиатура) — требуют интерактивного прогона, не статик-скрина.
- **Точные цифры контраста** (axe по живому dist) — a11y уже ≥90 (GO) на всех проверенных; нужны точные коэффициенты на пограничных.
- **Паритет длинных RU/UK-строк** построчно — горизонтальных переполнений нет (responsive GO), но не проверено покомпонентно.

_Проверено и НЕ требует действий: OG-превью (полный набор), PWA (manifest+sw), RatesBar fallback «—», empty-state поиска — реализованы корректно._

## 💪 Сильные стороны (не сломать при правках)

1. **Эталонная монетизация (§16/§18):** только `/go/{partner}` с `rel="sponsored nofollow noopener"`, статичные боксы, click-to-load виджеты (Trip.com/Google не грузятся до клика), ноль сторонних рекламных JS, одна аналитика (GA4).
2. **Строгий CSP** (`default-src 'self'`, `frame-ancestors 'none'`, `object-src 'none'`) + security-заголовки + теперь Cache-Control статики.
3. **3-язычное зеркало** (en — корень, ru→/ru/, uk→/uk/) с взаимными hreflang + x-default в head каждой страницы.
4. **Уникальные «живые» активы** — курс лари (NBG) и погода/море (open-meteo) в RatesBar с build-time снапшотом, fallback «—» и клиентским догрузом в idle.
5. **Резкий hero** — AVIF до 1600px (per-image quality) + webp-фолбэк через `<picture>`, `font-display: swap` + preload дисплей-шрифта; ротация кадра каждые 3 ч.
6. **Schema-база** — Organization+WebSite с `@id` site-wide, BreadcrumbList, TouristAttraction/Restaurant; авто-год в «денежных» title.
7. **Доступность в основе** — фокус-ловушка+Esc в лайтбоксе, `prefers-reduced-motion`, skip-link (`:focus-visible`), focusout закрывает меню, осмысленные `aria-labelledby`.
8. **Дизайн-токены — единый источник** (`tokens.css`), без хардкода hex в компонентах; шкала §10.
9. **Сигналы доверия** — метка «Проверено · дата», бейдж «Иллюстрация» у жанровых фото еды, золотая медаль качества (опт-ин); прозрачность партнёрских размещений.
10. **Жёсткие гейты** — `npm run qa` (GO/NO-GO: check/build/test/links/lint + статический аудит dist) + `check-enums/parity/photos/interlinks` + `qa:browser` (mobile-Lighthouse). PWA (оффлайн/установка).
