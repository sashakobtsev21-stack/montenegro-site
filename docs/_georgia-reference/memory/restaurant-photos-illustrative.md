Фото-беклог ресторанов `/eda/` закрыт (2026-06-17, коммиты `6d8fe64`→`fd9908b`): все 21 живой ресторан с обложкой.

**Реальность:** фото САМИХ заведений (фасад/интерьер конкретного ресторана) в легальном бесплатном доступе почти НЕ существует — по 21 ресторану Openverse (Flickr CC + Wikimedia) дал фото заведения только для 1 (vino-underground). Это не пробел поиска.

**Решение владельца (2026-06-17):** для остальных обложка = релевантное **жанровое CC-фото блюда/кухни** (хачапури, хинкали, оджахури, пхали, вино, морепродукты, веган, кофе, шаурма, чурчхела) с ОБЯЗАТЕЛЬНЫМ видимым **бейджем «иллюстрация»** — чтобы не выдать за снимок места (честность §15).
- Механика (исходно): поле `coverIllustrative: boolean` → RestaurantCard рисовал бейдж `.rcard__illus` (i18n `eda.illustrativeBadge`). `coverCredit` (автор+лицензия+источник) обязателен. Каждое фото — один раз.
- **ОБНОВЛЕНО 2026-06-18 (коммит `77af832`, решение владельца):** видимый бейдж «иллюстрация»/«Ілюстрація»/«Illustration» УБРАН со всех карточек — удалены рендер `.rcard__illus`, проп `illustrative`, передача из EdaDirectory/CityFoodPage, CSS; из `alt` обложек (ru/uk/en, 60 файлов) убраны пометки «(иллюстрация…/cuisine illustration)». Обложки = по-прежнему жанровые CC-фото блюд (НЕ фото самих заведений), но **без видимой пометки**. `coverIllustrative` (схема/frontmatter) и i18n `illustrativeBadge` оставлены спящими. **Бейдж не возвращать** без нового решения; `coverCredit` обязателен по-прежнему.

**Источники легальных фото (расширено):**
- `scripts/openverse-candidates.mjs "query"` — Openverse (Flickr CC + Wikimedia, фильтр коммерч-CC: cc0/by/by-sa/pdm).
- **Unsplash подключён (2026-06-17):** ключ владельца в **`.env` (gitignored — НЕ коммитить; `.gitignore` += `.env`)**, читается скриптами. `scripts/unsplash-candidates.mjs` (поиск) + `scripts/build-unsplash.mjs` (ОБЯЗАТЕЛЬНЫЙ download-trigger по API Guidelines → webp ≤200КБ). Атрибуция «Photo: Имя / Unsplash». **9 ресторанов общих тем** (кофе/вино/рыба/веган/меззе/шаурма/хинкали/хачапури/стол) переведены на Unsplash — качество выше. **Грузинская специфика** (лобиани/оджахури/пхали/чурчхела/шкмерули/застолье + cafe-littera) осталась на Flickr CC/Wikimedia: на Unsplash по ним пусто или fuzzy-мусор («georgian» = штат Джорджия, США). vino-underground — реальное фото заведения.
- Грабли: имя автора с апострофом (Keesha''s) в YAML single-quoted экранировать `''`; base raw Unsplash берёт `?w=...` (на URL без query). Сток как «своё» фото нельзя; всё с атрибуцией. См. [[content-photos-mandatory]], [[photo-pipeline-hybrid]], [[qa-gate]].
