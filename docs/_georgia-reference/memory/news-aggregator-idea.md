**Задумка владельца (2026-06-17):** по команде собрать новости с источников, обработать и выложить. Главная боль — **мусор**.

**Развилки закрыты владельцем (2026-06-19):**
- **Модель:** Claude Code **skill `/news`** — я гоняю конвейер своими инструментами (WebSearch/WebFetch + фото-пайплайн Unsplash/Commons + переводчики ru→uk→en + `qa`). НЕ headless-скрипт без LLM.
- **Публикация:** команда → батч → **превью владельцу → «ок» → публикация**. НЕ слепой автопостинг (по сути 1 команда + «ок»).
- **Тематика (= фильтр мусора):** **только тревел-релевантное** — въезд/визы/границы, авиа/аэропорты, цены/валюта/налоги туристу, погода/ЧС/безопасность, открытия-закрытия мест/транспорта, крупные события/фестивали. Политика/криминал/спорт/реклама/кликбейт — отсев.
- **Ритм:** дайджест за **24 ч, топ 3–5**; запускает владелец (раз в день/по желанию). Полный автопилот по расписанию = фаза 2 (GitHub Action + Claude через API, отдельная инфра/бюджет).
- **Источники (старт, владелец подтверждает/дополняет):** Agenda.ge, Civil.ge, OC Media, DFWatch, Georgia Today, Interpressnews EN; Google News RSS («Georgia travel/airport/visa/border/tourism»); офиц. — аэропорты (TAV), Georgian Airways, МВД/погранслужба, Нацбанк.

**Полное ТЗ — в `ROADMAP.md` → «БУДУЩИЕ КРУПНЫЕ ДОРАБОТКИ».** Ключевое при реализации:
- Правило 4: рерайт фактов с сохранением источника (`sources`), без галлюцинаций.
- Копирайт/ToS §15: чужие новости не копировать дословно — свой пересказ + ссылка.
- Фильтр мусора (главная боль): жёсткий отсев по релевантности (Грузия/тревел), дедуп, отсев рекламы/кликбейта.
- Фото обязательно ([[content-photos-mandatory]]), пара ru+uk (+en) авто-переводом.
- **НЕ слепая автопубликация:** «1 клик» = собрать+отфильтровать+черновики (`draft:true`) → владелец одобряет → публикация. Свежесть/ретеншн уже есть ([[news-deploy-hook-pending]]).
- Связан с агентом `research-collector`. Источники: RSS грузинских изданий + Google News RSS + офиц. каналы.

Как публиковать новости СЕЙЧАС (ручной процесс): добавить пару `ru+uk` в `src/content/articles/*/<slug>.md` (`category: 'novosti'`, `publishedAt`=дата, `cover` обязателен, `draft:false`) → push в main → Cloudflare соберёт. Или просто попросить меня.
