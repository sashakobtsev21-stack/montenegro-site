Стратегическое направление Georgia Guidebook (согласовано с владельцем 2026-06-20 после анализа `docs/GROWTH-MONETIZATION-ANALYSIS-2026-06-20.md` + `ROADMAP-GROWTH-2026.md`).

**Куда движемся:**
- Приоритет №1 — **ru/uk-ядро** (туристы РФ/КЗ/BY/UA). Сайт от него НЕ уходит. Это **НЕ разворот на запад**.
- **EN/номад — ВТОРОЙ, дополнительный рынок** (экспаты/номады в Грузии), добавляется ПОСЛЕ укрепления ядра, пилотом. «Запад = выше оплата за рекламу» касалось МЕДИЙНОЙ рекламы — от неё отказались (A4=нет); ценность запада теперь = дорогие/рекуррентные партнёрки (страховка номадов/Wise/релокация) + позже премиум-сети (нужна доля Tier-1).
- 3 вектора: (1) глубина+доверие на ядре через личный опыт резидента (владелец в Батуми) — анти-AIO/E-E-A-T; (2) деньги: дописать готовые денежные страницы + размазать доход с Trip.com (≈84%); (3) новые рынки бонусом: EN-слой + прямые размещения локалов Батуми (`/reklama/`).

**Решения владельца (2026-06-20):** A1 ✅ опыт пилотом на Батуми · A2 ✅ EN/номад пилот 3–4 статьи · A4 ✅ нет медийной рекламы (только партнёрка) · A5 ✅ `/reklama/` для локалов с интро-ценами · A3 ⬜ события отложены · A6 страховка ru/uk = ждём EKTA (не блокер).

**GSC/аналитика подключены 2026-06-20** (старт ~5 кликов/183 показа за 28 дней) → через 2–4 недели приоритизировать очередь по реальному спросу.

**Чего НЕ делаем:** медийную рекламу; разворот ru/uk→en; генерёнку без фактуры.

**Why:** владелец спросил «куда движемся» и был риск понять анализ как «бросаем русских, уходим на запад» — это НЕ так; зафиксировано, чтобы не дрейфовать.
**How to apply:** новый контент — по обновлённому `docs/CONTENT_PLAN.md` (§0); порядок исполнения — `ROADMAP-GROWTH-2026.md` (раздел «Операционные принципы»). Связано: [[update-docs-after-each-change]], [[affiliate-partners]], [[english-3rd-language]], [[content-photos-mandatory]].
