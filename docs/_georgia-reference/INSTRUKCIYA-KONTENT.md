# Инструкция по контенту — для владельца (просто, по шагам)

_Это «для чайников»: что делаешь ты, что делаю я, по дням. Технику (npm, код, деплой) делаю я — тебе в это лезть НЕ нужно. Подробная версия для ассистента — `CONTENT_GUIDE.md`; план тем — `docs/CONTENT_PLAN.md`; куда движемся — `ROADMAP-GROWTH-2026.md`._

---

## 0. Запомни главное (роли)

**Ты НЕ пишешь статьи и НЕ запускаешь ничего технического.**

- **Ты:** даёшь тему или факт → смотришь готовое → говоришь «ок».
- **Я (ассистент):** пишу (рус + укр + англ), беру фото, проверяю, публикую, обновляю сайт.
- **Сайт обновляется сам** после публикации (я делаю push → сайт пересобирается за пару минут).

## 1. Как «запустить» — это просто фраза мне

Никаких кнопок. Чтобы началась работа, пишешь мне одну из фраз:

- **«Идём по плану»** — беру следующую тему из очереди и пишу.
- **«Пиши про вино Кахетии»** (или любую тему) — пишу конкретную.
- **«Вот факт: в Батуми …»** — превращаю твой факт в статью (самое ценное — твой эксклюзив).

Дальше работаю я и приношу результат на «ок».

## 2. Неделя по шагам (шаблон; не обязаловка каждый день)

**🗓 Понедельник (15 минут) — твой единственный «рабочий» момент:**
1. Открой Google Search Console → раздел «Эффективность».
2. Посмотри 2 вещи: какие **запросы** появились и какие **страницы растут**. (Непонятно — пришли скрин, объясню.)
3. Напиши мне: **«идём по плану»** или **«на этой неделе пиши про <тему>»**.
4. Есть местные факты/фото — кинь мне.

**🗓 Вторник–четверг — работаю я:**
- Пишу статьи. Ты ничего не делаешь. Понадобится твой факт — спрошу.
- Приношу **превью** (текст/ссылку) → ты смотришь.

**🗓 Когда смотришь превью — проверь 3 вещи (30 секунд):**
- ✅ **Факты верны?** (цены/места/даты — знаешь, что не так → скажи)
- ✅ **Фото нормальные**, по теме?
- ✅ **Звучит по-человечески**, не «робот»?
- Ок → пишешь **«ок, публикуй»**. Не ок → **«поменяй вот это…»**.

**🗓 Пятница — публикация:**
- Публикую одобренное, сайт обновляется сам. Готово.

> Реально твоего времени: ~15 мин в понедельник + пара «ок» в течение недели. Всё.

## 3. Что писать (очередь — я веду, тебе помнить не надо)

Сейчас по порядку:
1. **Углубить гайды еды** (Тбилиси / Батуми / Кутаиси) — уже есть, делаю содержательнее.
2. **«Вино Кахетии»** — мост в винные туры (деньги).
3. **Цена/часы + факты** на топ-достопримечательностях и страницах Батуми — защита от Google AI (AI Overviews).
4. **Английские статьи для экспатов** (1% налог, work permit 2026, стоимость жизни) — бонус-рынок.

Полный список — `docs/CONTENT_PLAN.md`.

## 4. Что проверяю Я (тебе НЕ надо)

Перед каждой публикацией автоматически прогоняю: сборку сайта, проверку ссылок, вес фото (≤200 КБ), пары языков (рус+укр), разметку для Google → вердикт **GO**. Не GO — не публикую. Тебе вникать не нужно.

---

## 5. Как давать факты и фото (формат свободный)

Просто пиши как есть, например:
- «Лучшие хинкали в Батуми — <место>, адрес …, ужин на двоих ~40 лари, проверял в июне».
- «Бар X на бульваре закрылся».
- Фото: кинь файлы + скажи, где снято. (Свои фото → метка «Проверено» и плюс в Google.)

Я сам оформлю правильно и по правилам.

---

## 6. Чего НЕ делать

- ❌ **Не выдумывать** цены/факты «на глаз» — Google за это наказывает. Не уверен → скажи, помечу «уточняйте».
- ❌ Не лезть в код / npm / git — это моё.
- ❌ Не публиковать половину пары (только рус без укр) — я слежу за этим.
