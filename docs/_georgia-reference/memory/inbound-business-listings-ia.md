2026-06-16: к владельцу начали проситься локальные бизнесы для размещения — кофейни (в т.ч. одна с велопрокатом и киновечерами), квизы, лаундж-бары, клубы, столовые, рестораны, риелторы, клининги, «многие другие». Согласована переработка IA и подхода к монетизации.

**Ядро:** входящих сортируем по ИНТЕНТУ пользователя, не по типу бизнеса → 3 уже существующих раздела:
- Проголодался → **Еда** (кофейни, столовые, рестораны, бары про кухню).
- «Куда сходить» → **Развлечения** (клубы, лаундж-бары, квизы, киновечера, велопрокат + готовые казино/пляжи/парки).
- Живёт в Грузии, нужна услуга (сегмент C, §2) → **Релокация → Услуги** (риелторы, клининг, переезд, ремонт, документы, переводчики).

**Решения владельца (2026-06-16):**
1. Монетизация — **спонсор-карточки в готовых разделах** (НЕ отдельный платный каталог). Платное = карточка наверху с **золотой рамкой как ЕДИНСТВЕННЫМ маркёром** (бейдж/подпись владелец убрал 2026-06-16 ради чистоты вида — «только рамка»); курируемый >4★ бэкбон бесплатный. ⚠️ Я отметил риск: рамка без слова не читается посетителем как реклама (раскрытие рекламы / Google) — вернуть текст-подпись, если станет важно. NB: SPEC §8.6/§15 пока требуют бейдж «Партнёр» — при сборке согласовать SPEC с решением «только рамка».
2. **«Развлечения» — на главную** (5-я плитка `PRIMARY_TILE_KEYS`) + подкатегории-фильтры: ночная жизнь / афиша / активный / места + под-хабы по городам.
3. **Услуги — под-директория в `/relokatsiya/`** (НЕ новый раздел верхнего уровня, НЕ в Развлечениях).
4. **События (квизы/кино/концерты) — теги у заведений, БЕЗ календаря-афиши** (живая афиша дорога в поддержке и устаревает → бьёт по «свежести»-дифференциатору). Подтвердится спрос → лёгкая страница «Регулярные события в {городе}».

**Правила:**
- **Один бизнес — один канонический раздел + теги** (двери в др. разделы). Кофейня-с-велопрокатом-и-кино: дом = Еда, теги «велопрокат»/«киновечера» → всплывает и в Развлечениях. Один URL, без дублей (SEO/канибализация).
- **Фото от бизнеса** (с разрешения) брать частью сделки → закрывает потолок только-CC-фото и даёт реальные фото/право на бейдж.
- Цена реалистична (сайт молодой, трафик растёт): старт $20–50/мес или бартер за фото. Для услуг — мин. фильтр качества: мошенник бьёт по доверию ко всему сайту.
- Технически: самохостовый webp-баннер через `/go/`, без правки CSP; правило «одна аналитика, без сторонних виджетов» (CLAUDE §8) не нарушается.

**Контент-модель (правка SPEC §11, поля опциональные):** расширить `placeType` (+bar/club/quiz/cinema) и добавить `tags` в `restaurants`; новая коллекция `services` (рубрика/город/контакт/sponsored). Обобщить директорию мест.
