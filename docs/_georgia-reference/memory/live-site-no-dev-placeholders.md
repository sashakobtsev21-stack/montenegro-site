Сайт georgiaguidebook.com **уже публичный**. Владелец требует, чтобы внешний вид был «в рабочем состоянии», а контент добавляется постепенно.

**Why:** dev-надписи («добавим партнёрские ссылки», «демонстрация шаблона», «это демо-маршрут», «карточки-образцы», «(R3)/(R4)», TODO-комментарии) выглядят непрофессионально на живом сайте.

**How to apply:**
- Никаких служебных/«в разработке» формулировок в **рендеримом** тексте и в alt/aria. Пустые разделы — публичное «Раздел наполняется: скоро здесь появятся проверенные места…», а не «добавим/демо/шаблон».
- Партнёрки ещё не подключены (`partners.json` пуст) → НЕ писать «кнопка появится после подключения партнёрских сервисов»; вместо плейсхолдеров — польза (ссылка на профильный раздел) или нейтральный публичный текст.
