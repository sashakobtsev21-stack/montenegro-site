# Montenegro Guidebook

Контентный сайт-справочник по Черногории на трёх языках (**EN — на корне `/`**, RU — `/ru/`, UK — `/uk/`): достопримечательности, города, еда, маршруты, транспорт, аренда авто, страховки, развлечения, релокация, новости. Диверсифицированная партнёрская монетизация только через `/go/{partner}`.

> **Статус:** bootstrap. Это форк рабочего движка **Georgia Guidebook** (`georgia-guidebook`):
> система сохранена целиком, грузинский контент снят, коллекции пустые (сайт собирается
> пустым по дизайну). **Контент и перемоделирование под Черногорию** (enums регионов/
> городов, SPEC, тексты, гео Tier-1, EN-first) — **следующий шаг**. Грузинские шаблоны —
> в `docs/_georgia-reference/`.

> **Домен** `montenegroguidebook.com` — **плейсхолдер** (ещё не куплен); реальный задаётся на деплое.

## Стек

- **Astro 6** (полная статика SSG, без React) + **Content Collections**
- **Tailwind CSS 4**; дизайн-токены — только из `src/styles/tokens.css`
- Хостинг **Cloudflare Workers (Static Assets)** + Worker-роут `/go/` (`worker/index.ts`)
- Изображения webp/avif ≤ 200 КБ через предгенерацию вариантов (`npm run build:covers`)
- PWA (установка + оффлайн); одна аналитика

## Команды

| Команда | Что делает |
|---|---|
| `npm run dev` | dev-сервер |
| `npm run build` | прод-сборка (обязана быть без ошибок/предупреждений) |
| `npm run check` | `astro check` — типы и схемы контента |
| `npm run lint` / `npm run format` | ESLint / Prettier |
| `npm test` | контент-гейты: enums · parity (ru/uk/en) · photos · interlinks (≥2) |
| `npm run new -- <тип> <slug> --title "…"` | скаффолд материала (ru/uk/en + DoD-чеклист) |
| `npm run qa` | финальный GO/NO-GO: чистая сборка + гейты + статический аудит `dist/` |

## Документация

1. **`SPEC.md`** — продукт, ниша, шаблоны, контент-модель (единственный источник правды по продукту).
2. **`CLAUDE.md`** — железные правила работы в репозитории.
3. **`CONTENT_GUIDE.md`** — гайд по созданию контента.
4. **`SETUP-GUIDE.md`** — подключение внешних агентов/плагинов (my_agents, ui-ux-pro-max) как плагинов Claude Code.
5. **`docs/memory/`** — проектная память (зеркало авто-памяти). **`docs/_georgia-reference/`** — архив движка-донора (шаблоны).

> Форкнут из движка `georgia-guidebook`. Внешний тулинг подключается как плагины Claude Code — см. **`SETUP-GUIDE.md`**.
