## 1. Что это за проект
**Georgia Guidebook** (georgiaguidebook.com) — двуязычный (ru+uk) контентный сайт-**справочник** по всей Грузии: достопримечательности, города, еда (директория проверенных мест), маршруты, аренда авто, страховки, новости.

- Монетизация — диверсифицированная партнёрка (экскурсии, жильё, аренда, страховки, eSIM) через `/go/{partner}`.

## 2. Стек, хостинг, деплой
- **Astro 6 + Content Collections, полная статика (SSG).** Без SSR/CMS/БД.

- **Деплой = push в `main`** (Workers Build: `npm run build` → `wrangler deploy`). Превью-деплои на ветках.
- Домен **georgiaguidebook.com подключён** (Custom Domain `www.`), веб-аналитика Cloudflare включена.
