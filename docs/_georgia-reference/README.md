# `_georgia-reference` — архив проекта Georgia Guidebook (ШАБЛОНЫ)

> ⚠️ Это **справочный архив** материалов рабочего движка Georgia Guidebook, на котором
> форкнут montenegro-site. Содержимое относится к **Грузии** и оставлено здесь как
> **шаблоны/образцы** для следующего шага — перемоделирования под Черногорию.
> Это **НЕ актуальная документация** montenegro-site и в продакшен/сборку не попадает.

## Что здесь лежит и как использовать

| Файл / папка | Что это (по Грузии) | Как переиспользовать под Черногорию |
|---|---|---|
| `CONTENT_PLAN.md` | План тем/разделов контента | Образец структуры плана — переписать под гео и спрос Черногории |
| `GROWTH-MONETIZATION-ANALYSIS-*.md` | Анализ роста и монетизации | Методология та же — пересчитать рынок/партнёрки под Черногорию |
| `ROADMAP.md`, `ROADMAP-GROWTH-2026.md` | Роадмапы доработок | Шаблон роадмапа фаз R0–R6 |
| `PROGRESS.md`, `HANDOFF.md` | Журнал статуса и снимок «где остановились» | Формат ведения прогресса — начать с чистого листа |
| `AUDIT.md` | Сводный аудит сайта | Чек-лист аудита как образец |
| `TODO-affiliates.md` | Бэклог по партнёркам | Список партнёрок — пересобрать под доступные в Черногории |
| `INSTRUKCIYA-KONTENT.md`, `KALENDAR.md` | Инструкция/календарь для владельца | Формат рабочего процесса |
| `briefs/` | Брифы статей (вход конвейера) | Образцы брифов — писать новые под Черногорию |
| `memory/` | Проектная память Georgia Guidebook | История решений движка; новая память — в `docs/memory/` |

## Системные доки (НЕ здесь — в корне репозитория, актуальны)

`SPEC.md`, `CLAUDE.md`, `CONTENT_GUIDE.md`, `SETUP-GUIDE.md` остались в корне как
**рабочие** документы движка. Грузинская специфика внутри них (enums регионов/городов,
тексты, домен) перемоделируется под Черногорию на следующем шаге, а не в этом архиве.
