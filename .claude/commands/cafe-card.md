Создай карточку заведения для раздела «Еда» на RU+UK+EN по правилам
CONTENT_GUIDE.md. Данные заведения (название, описание, адрес, часы, фото)
от меня: $ARGUMENTS

1. Если чего-то критичного не хватает (название/адрес/часы) — спроси.
2. Создай три версии (одинаковый slug) в коллекции раздела «Еда» —
   `restaurants` (НЕ `articles`; поля карточки заведения см. CONTENT_GUIDE.md §12).
3. Фото: используй те, что я приложил/положил в public/images/<slug>/.
   Если фото нет — оставь <!-- ФОТО: вставлю сам -->. НЕ бери фото из
   интернета (это присланные заведением материалы, чужие права).
4. Текст: короткое живое описание (не реклама-простыня), адрес, часы,
   район. UK не калька, EN для иностранца.
5. Внутренние ссылки: на родственные статьи (рестораны города, что
   посмотреть рядом) — мин. 2, на язык версии.
6. Прогони `npm run qa`, добейся GO. Покажи результат. Коммить после «ок».
